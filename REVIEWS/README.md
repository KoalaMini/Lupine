# REVIEWS/ 审查报告

本目录存放**评估器**产出的审查报告。

## 命名规范

`{功能名称}-v{版本号}-{YYYYMMDD}-{类型}.md`

- `v{版本号}`：审查轮次版本，如 `v1.0`、`v1.1`
- `YYYYMMDD`：审查日期
- `{功能名称}`：中文功能名称，与对应 SPECS/PLANS 文件一致
- `{类型}`：`plan`（方案审查）或 `code`（代码审查）

## 示例

- `用户认证-v1.0-20260513-plan.md`
- `用户认证-v1.0-20260513-code.md`
- `用户认证-v1.1-20260515-plan.md`

## 内容要求

见 `AGENTS.md` 评估器章节。
