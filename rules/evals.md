# Lupine 评估标准

## 安全门禁（阻断）
- [ ] 密码/密钥/token 绝不硬编码
- [ ] 敏感信息不出现在日志或响应中
- [ ] SQL 注入防护
- [ ] 认证中间件覆盖所有需鉴权的路由

## 质量门禁（阻断）
- [ ] 新增代码测试覆盖率 ≥ 80%
- [ ] 圈复杂度 ≤ 15
- [ ] 不引入新的 lint error

## 架构门禁（警告）
- [ ] handler 层不含业务逻辑
- [ ] service 层不含 HTTP 细节
- [ ] 依赖方向是否正确

## 计划评估①扩展

### 多源一致性检查
- [ ] 同一规范在不同文档中的表述是否冲突（如 skill SKILL.md vs agents.md）
- [ ] 实现是否遵循最新版本的规则文件

## 代码评估②扩展

### 冒烟覆盖检查
- [ ] 常规路径测试通过
- [ ] 分支/异常路径测试通过（含所有交互选项）
- [ ] 错误路径测试通过（文件缺失、权限不足等）
