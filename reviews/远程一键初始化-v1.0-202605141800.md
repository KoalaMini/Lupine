## 5. 发现的问题

### 5.1 建议优化（非阻断）

**问题**：`install.sh` 中 `sed` 替换使用 `/` 作为分隔符

```bash
sed -i.bak "s/{项目名称}/$project_name/g" "CLAUDE.md"
```

如果项目名包含 `&` 等 sed 特殊字符，替换结果可能不正确。

**建议**：使用 `|` 作为分隔符，避免与项目名中的特殊字符冲突：

```bash
sed -i.bak "s|{项目名称}|$project_name|g" "CLAUDE.md"
```

### 5.2 可接受（不修改）

- **产物包含 scripts/build-release.sh**：install.sh 只复制 assets/ 文件，用户不会得到此脚本；仅影响手动解压 tarball 的用户
- **使用第三方 action `softprops/action-gh-release@v1`**：广泛使用的成熟 action，supply chain 风险可控
